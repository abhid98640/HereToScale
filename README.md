# Here to Scale

Growth agency website — design, development, and content workspace.

## Structure

```
HereToScale/
├── src/
│   ├── components/     # Reusable UI components
│   ├── pages/          # Page-level files (Home, About, Services, Contact, etc.)
│   ├── assets/
│   │   ├── images/     # Photos, backgrounds, hero images
│   │   ├── fonts/      # Custom fonts
│   │   └── icons/      # SVG icons and logos
│   └── styles/         # Global CSS / Tailwind / design tokens
├── public/             # Static files served as-is
└── docs/               # Copy, briefs, brand guidelines, sitemap
```
